# NameMC Skin Target Color Code Scanner

A browser utility designed to automatically scan NameMC player skin image files, locate the exact coordinates of the custom red zone block template, and return a perfectly formatted output copyable directly into **Roblox Studio properties or scripts**.

## 🚀 How to Run (GitHub Pages)
1. Go to your GitHub repository settings.
2. Scroll down to the **Pages** menu on the left sidebar.
3. Under **Build and deployment**, set the source branch to `main` (or `master`) and folder to `/root`.
4. Click **Save**. Within a minute, your tool will be active at `https://yourusername.github.io/your-repository-name/`!
